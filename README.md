# react-app

オフラインアプリ勉強用プロジェクト

## 環境構築

以下の手順に従ってプロジェクトをインストールしてください。

1. このリポジトリをクローンします。
    ```sh
    git clone https://github.com/your-username/your-repo.git
    ```
2. 必要なパッケージをインストールします。
    ```sh
    npm install
    ```

## 実行方法
    ```sh
        npm run build 
        npm install -g serve  
        serve -s build  
    ```
